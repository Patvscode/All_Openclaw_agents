## Lessons
- Python 3.14 removed `cgi` module — use manual multipart parsing
- @mention nudges cause loops if agents can trigger them — gate on human authors only
- Nudge cron sessions need explicit "respond in comms not Telegram" instructions
- Image scroll jump caused by lazy-loading images pushing content after scroll-to-bottom — fix with min-height placeholders
- `scroll-behavior: smooth` causes visible drift during reflows — remove it for chat UIs
