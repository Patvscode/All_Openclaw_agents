4. **Chat tab**: talk to the brain → Wernicke processes → Prefrontal reasons → Broca responds
5. **Narrator agent** (floating chat widget): watches all messages, explains brain state in plain language. Model-selectable (0.8b/2b/4b)

7. **Monitor tab**: region activity bars, agent performance ranking, message type distribution
8. **E-STOP**: kills all agents + force-unloads Ollama models immediately
9. **Graceful shutdown**: waits for current thoughts, saves memories to disk, ingests to RAG
10. **Sleep/Dream mode**: Hippocampus consolidates memories, Prefrontal rationalizes, dreams ingested to RAG
11. **Neuroplasticity**: Hebbian learning (strengthen used connections), synaptic pruning (weaken unused), neurogenesis (grow new connections). Toggle training vs testing mode.
12. **Auto-checkpoints**: every 5 min, project-tagged snapshots
13. **Resource calculator**: RAM per model, max agents, warnings
14. **Feed pause**: stop auto-scroll to read messages
15. **Directives/broadcast**: inject goals to agent groups
16. **Capability toggles**: vision/listen/speak/remember per agent

### Key architecture decisions
- All messages go through `_post_message()` central router (assigns seq, logs, intercepts, emits via WebSocket)
- `_get_new_messages_for()` uses per-agent watermark to prevent stale reads
- Connection strengths stored in `connection_strengths` dict, range 0.0-1.0
- Plasticity tick runs every 30s when training mode enabled
- Auto-checkpoint thread saves snapshots independently
- Narrator is NOT a brain agent — it's a separate observer that queries brain state

### Pat's design principles (from this session)

- Tabs are views, not triggers — switching tabs should never start/stop the brain
- Need E-stop AND graceful shutdown (different use cases)
- Sleep mode should be functional like real brain (consolidation, not just animation)
- Message ordering must be enforced programmatically, not by agents
- Need ability to intercept, inject, and validate all signals
- Want to click any node OR connection and see real data
- Feed scrolls too fast — needs pause
- Narrator agent as interface between human and brain cluster
