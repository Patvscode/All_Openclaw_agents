## Session 4 (~09:00-12:20 EST) — Computer Use V2 Fix + Video Understanding + GPU Management

### Computer Use V2 — Root Cause Diagnosis & Fix
**Root cause of failures (from log analysis of 10 recent sessions):**
1. **#1 killer: Google CAPTCHA** — Automated Playwright hitting Google → reCAPTCHA → agent stuck looping 15 times
2. **Selector timeouts** — Planner outputs CSS selectors that don't exist (Page.fill: Timeout 5000ms)
3. **Poor error recovery** — Errors in history but planner repeats same mistake
4. **Premature done** — Agent says "done" on step 1 with no data
5. **Missing params** — `goto` with no URL, `click` with no target

**Fixes applied (via sub-agent):**
- Default search engine → DuckDuckGo (no CAPTCHA)
- Fallback to Bing if DuckDuckGo fails
- Error recovery: "LAST ACTION FAILED" warning to planner
- Min 3 steps enforced before allowing "done"
- Param validation before executing actions
- Type fallback: if CSS selector fails, fall back to keyboard typing

**New features:**
- POST /api/sessions/continue — resume from where a session stopped
- POST /api/sessions/research — auto-research why a task failed
- GET /api/sessions/history — browse all past sessions
- GPU management panel (GET/POST /api/gpu/*)
- Continue + Research Fix buttons on session cards
