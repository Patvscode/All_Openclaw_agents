# 2026-03-10 — Isaac Lab Hide-and-Seek Build Day

## Major Accomplishment
Built a complete 2v2 hide-and-seek multi-agent RL environment in NVIDIA Isaac Lab from scratch in one session. Replicating OpenAI's "Emergent Tool Use" (2019) paper on the DGX Spark.

## What Was Built

### Isaac Lab Environment (Isaac-Hide-And-Seek-Direct-v0)
- DirectMARLEnv subclass with 4 agents (2 hiders, 2 seekers)
- 24m walled arena with 4 movable boxes + 2 ramps
- Zero-sum reward via line-of-sight (2D ray-AABB intersection)
- Prep phase (40%) with frozen seekers + Seek phase (60%)
- 37-dim observations, 3-dim continuous actions
- IPPO and MAPPO training configs for SKRL

### Infrastructure
- CLI commands: isaac-train, isaac-visual, isaac-stop, isaac-status (~/bin/)
- Mobile dashboard: hub/32-isaac-training/ with live stats
- Monitor API on port 8104 (Flask, systemd)
- VNC streaming via x11vnc + noVNC (ports 5900, 6080)
- TensorBoard on port 6006

### Training
- Verified on GPU (cuda:0), 256 parallel envs, ~15-20 it/s
- Training ran to 250K+ steps by evening
- Checkpoints saving to IsaacLab/logs/skrl/hide_and_seek/

## Key Technical Lessons
- `RigidObjectCfg` is in `isaaclab.assets`, NOT `isaaclab.sim`
- SKRL configs need `agent.experiment.directory` or KeyError
- GB10 unified memory means processes don't show in nvidia-smi memory column
- Isaac Sim Python 3.11 != system Python 3.14 — always use isaaclab.sh -p
- noVNC works great for mobile streaming of Isaac Sim viewport
- x11vnc -scale 1280x720 needed for mobile-friendly resolution
