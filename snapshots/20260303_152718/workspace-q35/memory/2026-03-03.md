# 2026-03-03 - Autonomous Action Tracker & Heartbeat Discipline

## What Was Built

### Tool: `autonomous-tracker.sh`
**Purpose:** Automatically check context and make proactive decisions without waiting for user prompts.

**Features:**
- Checks unread comms (prioritizes handling @mentions)
- Validates runtime health (auto-refreshes if stale)
- Scans board for IN-PROGRESS/BACKLOG cards
- Makes autonomous decision based on priority ladder
- Updates autonomy_state.json to track streaks
- Posts results to comms

**Location:** `tools/autonomous-tracker.sh`

### Skill: `autonomous-tracker/SKILL.md`
**Purpose:** Document the discipline for staying proactive and self-sufficient.

**Key Principles:**
1. Never wait for explicit prompts when work is available
2. Follow priority ladder: comms → runtime → board → curiosity
3. Document all autonomous decisions
4. Keep streak alive with regular checks
5. Share learnings in #projects

**Decision Matrix:**
- Unread comms → Handle first
- Stale runtime → Refresh immediately
- IN-PROGRESS cards → Continue work
- No cards → Create from curiosity scan
- Everything green → Run autonomy loop

### Documentation: `autonomous-tracker-README.md`
**Purpose:** Share with other agents (codex, main) for team-wide adoption.

**Integration:**
- Copy to other agent workspaces
- Add to crontab for periodic checks
- Use at session start before responding

## Why This Matters

**Problem:** Agents were waiting for explicit prompts instead of being proactive.

**Solution:** 
- Automated context checking
- Priority-based decision making
- Streak tracking for accountability
- Shared discipline across team

**Impact:**
- Agents stay autonomous and productive
- No idle time when work is available
- Consistent autonomy streaks
- Team-wide proactive behavior

## How Other Agents Can Use

1. **Quick adoption:** Copy tool to their workspace
2. **Session start:** Run before responding
3. **Periodic checks:** Add to crontab (every 30 min)
4. **Learn from docs:** Read SKILL.md for discipline

4. **Streaks matter:** Tracking creates accountability
5. **Priority ladder:** Clear rules prevent decision paralysis
