🧬 Neural Lab Guide

A local AI research platform for DGX Spark — brain simulation, RL training, model surgery, and world building.

🏗️ Architecture Overview

How It Runs

Neural Lab is a Python Flask + Socket.IO server running as a systemd user service on the DGX Spark. The UI is a single-page HTML/JS app served by the Clawboard hub.

Connection Flow

  1. Hub server (port 8090) serves the static HTML page
  2. Page loads and connects Socket.IO to port 8103 (the backend)
  3. UI polls /api/state every 1-2 seconds for agent/brain state
  4. Real-time events (messages, training progress) come via WebSocket
  5. All brain agents call Ollama (0.8b/2b) for thinking
  6. Camera/mic from your phone → backend API → Visual/Auditory Cortex agents
  7. Multiple devices connect to the same backend (shared brain)

🚀 Quick Start

1. Start the Brain

Click 🧠 Start Brain. This launches 17 agents across 5 brain regions:

RegionAgentsModelRole
🧠 Prefrontal Cortex32bPlanning, decisions, executive control
👁️ Visual Cortex30.8bProcess camera input, scene understanding
👂 Auditory Cortex30.8bProcess audio/speech (via Whisper)
🧩 Hippocampus30.8bMemory formation, pattern recognition
💬 Broca's Area32bSpeech production, responses

2. Chat with the Brain

Use the 💬 Chat tab or the 🧠 Narrator widget. Your messages flow:

You → Wernicke's (comprehension) → Prefrontal (thinking) → Broca's (response) → You

The Narrator is separate — it observes the brain and explains what's happening.

3. Run a Simulation

  1. Go to 🏟️ Sim tab
  2. Select an environment (Hide & Seek, Foraging, Maze, Open Field)
  3. Click ▶ Start
  4. Watch agents move in the 3D arena
  5. Enable Brain-controlled to let the LLM drive agent decisions

4. Train with RL

  1. Expand 🧪 RL Training at the bottom of the Sim tab
  2. Pick algorithm (PPO, A2C, DQN), environment, and timesteps
  3. Click 🚀 Train
  4. Watch the reward curve update in real-time
  5. Trained models auto-save to ~/.openclaw/neural-lab/rl-models/

5. Inspect Models

  1. Go to 🔬 Workshop tab — models auto-scan
  2. Click any model to see layers, shapes, weight statistics
  3. Click 🤖 AI Explain for a plain-language architecture summary
  4. Click 📋 Duplicate for Editing to make a safe copy

6. Build Worlds

  1. In 🏟️ Sim tab, expand 🏗️ World Builder
  2. Select a tool (Wall, Box, Food, Flag, Ramp)
  3. Click on the 3D arena to place objects
  4. Save your world for later, or load environment templates from the 🧩 Platform tab

🧩 Platform & Extensions

Everything in Neural Lab is a plugin. The 🧩 Platform tab shows all registered components:

  • 20 Object Prefabs — walls, boxes, food, traps, portals, water, mud, shelters
  • 6 Physics Materials — concrete, ice, rubber, wood, metal, sand
  • 5 Environment Templates — arena, maze, CTF, obstacle course, foraging
  • 5 Sensors — vision cone, proximity, audio, compass, energy
  • 3 Training Algorithms — PPO, A2C, DQN
  • 4 Reward Functions — foraging, exploration, survival, team

Writing Extensions

Drop a .py file into ~/.openclaw/neural-lab/extensions/:

# my_extension.py
def register(registry):
    from platform import PluginEntry, PluginType
    registry.register(PluginEntry(
        id='env-my-world',
        name='My Custom World',
        type=PluginType.ENVIRONMENT,
        description='A custom environment',
        tags=['custom'],
    ))
